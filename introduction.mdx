## Installation

<Tabs>
  <Tab title="script">
  Inside of your project import it using a script tag:

  ```js
  <script src="https://github.com/inputify/inputify-js/cdn/inputify.js" crossorigin="anonymous"></script>
  ```
  </Tab>
  <Tab title="minimal-script">
  Inside of your project import it using a minimal script tag:
  
  ```js
  <script src="https://github.com/inputify/inputify-js/cdn/inputify.min.js" crossorigin="anonymous"></script>
  ```
  </Tab>
</Tabs>

## Previewing Changes Locally (under developement)

You can run a web server free using bartender (also developed by inputify team)
```bash Terminal
npm i bartender
```

Than start a web server using:

```bash Terminal
bartender shot
```

## Troubleshooting

Here's how to solve some common problems when working with the CLI.

<AccordionGroup>
  <Accordion title="Inputify is not loading">
    Check the link or import it using another method
  </Accordion>
  <Accordion title="Inputify command not working">
    Check if you typed the command correct and check if the command is available in your script version, cause minimal script and normal script have different modules...
  </Accordion>
  <Accordion
    title="More Troubleshooting"
  >
    Please open a issue in [github](https://github.com/inputify/inputify-js) and we will to solve it.
  </Accordion>
</AccordionGroup>
