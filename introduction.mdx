## What is Inputify?
A **simple** and **powerful** javascript library to get keys inputs. With support for keyboards, MIDI, and controllers from almost all the consoles (or alternatives). Made with pure javascript to be fast and light ðŸ’ª.

## Installation

<Tabs>
  <Tab title="script">
  Inside of your project import it using a script tag:

  ```js
  <script src="https://github.com/inputify/inputify-js/cdn/inputify.js" crossorigin="anonymous"></script>
  ```
  </Tab>
  <Tab title="minimal-script">
  Inside of your project import it using a minimal script tag:
  
  ```js
  <script src="https://github.com/inputify/inputify-js/cdn/inputify.min.js" crossorigin="anonymous"></script>
  ```
  </Tab>
</Tabs>

## Previewing Changes Locally

Run the following command at the root of your documentation where `mint.json` is located. Mintlify will open at `localhost:3000`.

```bash Terminal
mintlify dev
```

Here's how to solve some common problems when working with the CLI.

  <Accordion title="Inputify is not loading">
